Pengenalan HTML & CSS

HTML (HyperText Markup Language) adalah bahasa dasar untuk membuat struktur halaman web. Sedangkan CSS (Cascading Style Sheets) digunakan untuk mengatur tampilan dan desain web agar lebih menarik.

Struktur Dasar HTML

Berikut adalah contoh struktur dasar HTML:

              <!DOCTYPE html>
              <html>
                <head>
                  <title>Judul Halaman</title>
                </head>
                <body>
                  <h1>Selamat Datang</h1>
                  <p>Ini adalah halaman web pertama saya!</p>

Cara Menggunakan CSS

Berikut adalah contoh penggunaan CSS dalam HTML:

              <style>
                body {
                  background-color: lightblue;
                }
                h1 {
                  color: white;
                  text-align: center;
                }
              </style>

Langkah Belajar HTML & CSS:

  1. Pelajari dasar-dasar HTML
  2. Pelajari elemen dan atribut HTML
  3. Pelajari dasar-dasar CSS
  4. Coba membuat halaman web sederhana
  5. Eksplorasi lebih dalam dengan layout dan responsivitas
